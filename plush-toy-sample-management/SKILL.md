# 毛绒玩具样品管理

## 用途

帮助用户完成毛绒玩具新品的完整估价流程，从材料成本到最终报价，并管理样品信息。

## 何时使用

当用户需要：
1. 估算新品毛绒玩具的成本
2. 生成报价单
3. 管理样品信息
4. 查询样品状态
5. 更新样品记录

## 估价流程

### 第一步：收集基础信息
询问并确认：
- 产品名称
- 规格型号（尺寸）
- 生产数量

### 第二步：确认材料清单
根据产品类型确认所需材料：
- **主料**：各种布料（玫红毛料、亮布、拉毛布、彩色头毛等）
- **辅料**：眼睛、鼻子、玻璃珠等配件
- **填充物**：充棉材料（按克重）
- **其他**：魔术贴、拉链、标签等

### 第三步：询问材料单价
必须明确价格类型：
- **散剪价**：小批量，单价高
- **整卷价**：大货，单价低（需整卷采购）

### 第四步：确认加工工序
按顺序确认是否需要：
1. **复合**（弘辉复合-宋总）
2. **激光**（刘旭-杨总）
3. **绣花**（朱师傅-杨总）
4. **车缝**（冯杰/邓总/赵总）
5. **充棉**（张总/徐凯）
6. **手工**（封口）
7. **质检**
8. **包装**
9. **物流**

### 第五步：询问加工单价
记录每个工序的单价。

### 第六步：计算材料成本
使用公式：用量 × 单价 × (1+损耗率)

**损耗标准（固定）**：
- 布料：10%
- 配件（眼睛/鼻子等）：5%
- 充棉材料：0%

### 第七步：计算加工成本
各工序单价之和 × 数量

### 第八步：汇总总成本
材料成本 + 加工成本 + 物流 + 其他

### 第九步：生成报价建议
- 成本价
- 建议报价（通常加10-20%利润）
- 订金30%
- 尾款70%

### 第十步：录入样品管理表
将估价结果录入飞书样品管理表。

## 报价格式

```
【材料成本】
材料1：用量 × 单价 = 金额（损耗说明）
材料2：用量 × 单价 = 金额（损耗说明）
...
材料成本合计：XXX元（X.XX元/个）

【加工成本】
工序1：单价 × 数量 = 金额
工序2：单价 × 数量 = 金额
...
加工成本合计：XXX元（X.XX元/个）

【成本汇总】
总成本：XXX元
成本单价：X.XX元/个

【报价建议】
建议报价（X%利润）：X.XX元/个
合计金额：XXX元
订金30%：XXX元
尾款70%：XXX元
```

## 常用参考价

| 工序 | 参考单价 | 供应商 |
|------|---------|--------|
| 复合 | 咨询宋总 | 弘辉复合 |
| 激光 | 0.5-0.8元/个 | 刘旭（杨总）|
| 绣花 | 0.4-0.7元/个 | 朱师傅（杨总）|
| 车缝 | 1.5-3元/个 | 冯杰、邓总、赵总 |
| 充棉+手工 | 2-3元/个 | 张总、徐凯 |
| 物流 | 0.2-0.3元/个 | - |

## 检查清单

每次估价必须确认：
- [ ] 材料清单完整（主料+辅料+配件+填充物）
- [ ] 材料单价明确（散剪价/整卷价）
- [ ] 损耗比例正确（布料10%、配件5%、棉0%）
- [ ] 加工工序完整（复合→激光→绣花→车缝→充棉→手工→质检→包装）
- [ ] 物流费用包含
- [ ] 包装费用包含
- [ ] 利润比例合理（10-20%）
- [ ] 付款方式明确（订金30%）

## 电商利润计算

当用户需要进行电商销售利润分析时，参考 [references/ecommerce-profit.md](references/ecommerce-profit.md)。

**关键参数：**
- 电商成本：1.8元/个（快递+打包+仓储+纸箱）
- 组合发货：2只+0.1元，3只+0.2元
- 平台扣点：5%
- 发货前退款：不计收入、无成本
- 发货后退货：按产品丢弃计算损失
- 发货后退货基数：按成交订单总数计算

## 样品管理表

飞书多维表格：样品管理表（tblDXFCNzEHlcoHZ）

### 录入字段
- 样品编号（自动生成）
- 样品日期
- 客户
- 产品名称
- 规格型号
- 材料清单
- 材料成本
- 打样成本
- 总成本
- 估算价格
- 样品状态
- 关联订单
- 备注

### 状态流转
1. 打样中（初始状态）
2. 已完成（打样完成）
3. 客户确认（客户确认价格）
4. 放弃（取消打样）

## 注意事项

1. **必须按检查清单逐项确认**，避免漏项
2. **材料单价必须明确是散剪价还是整卷价**
3. **损耗比例固定不可变**：布料10%、配件5%、棉0%
4. **加工工序必须确认完整**，不能漏掉任何环节
5. **估价完成后立即录入样品管理表**
