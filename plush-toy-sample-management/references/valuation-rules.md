# 毛绒玩具样品估价规则与流程

## 一、估价方式

### 方式一：散剪价（小批量/打样）
- 布料：散剪单价 + 10%损耗
- 配件：单价 + 5%损耗
- 适合：样品、小批量订单（通常<500个）

### 方式二：大货价（批量生产）
- 布料：整卷单价 + 10%损耗
- 配件：单价 + 5%损耗（或根据实际）
- 适合：1000个以上批量订单

### 方式三：无损耗估算（理想状态）
- 所有材料：按实际用量，不加损耗
- 用于：成本底线核算、利润分析

## 二、估价流程（10步）

```
第一步：收集基础信息
    ↓ 产品名称、规格、数量
第二步：确认材料清单
    ↓ 主料、辅料、配件、填充物
第三步：询问材料单价
    ↓ 散剪价？整卷价？
第四步：确认加工工序
    ↓ 复合？激光？绣花？车缝？充棉？手工？
第五步：询问加工单价
    ↓ 每个工序的单价
第六步：计算材料成本
    ↓ 用量 × 单价 × (1+损耗率)
第七步：计算加工成本
    ↓ 各工序单价之和 × 数量
第八步：汇总总成本
    ↓ 材料 + 加工 + 物流 + 其他
第九步：生成报价建议
    ↓ 成本价 + 利润比例
第十步：录入样品管理表
```

## 三、材料成本检查项

### 必须确认的材料
- [ ] **主料**：各种布料（含门幅、单耗）
  - 玫红毛料、亮布、拉毛布、彩色头毛等
- [ ] **辅料**：眼睛、鼻子、玻璃珠等配件
- [ ] **填充物**：充棉材料（按克重）
- [ ] **其他**：魔术贴、拉链、标签等

### 损耗标准（固定）
| 材料类型 | 损耗比例 | 说明 |
|---------|---------|------|
| 布料 | 10% | 裁剪损耗，固定 |
| 配件（眼睛/鼻子等）| 5% | 损坏/丢失，固定 |
| 充棉材料 | 0% | 按实际重量，无损耗 |
| 其他辅料 | 3-5% | 视情况而定 |

### 价格类型确认
- **散剪价**：小批量，单价高
- **整卷价**：大货，单价低（需整卷采购）
- 必须明确：当前报价用哪种价格

## 四、加工成本检查项

### 必须确认的工序
- [ ] **复合**：是否需要复合（弘辉复合-宋总）
- [ ] **激光**：激光下料费用（刘旭-杨总）
- [ ] **绣花**：绣花费用（朱师傅-杨总）
- [ ] **车缝**：平车加工费用（冯杰/邓总/赵总）
- [ ] **充棉**：充棉材料 + 充棉手工（张总/徐凯）
- [ ] **手工**：封口手工费用
- [ ] **质检**：质检费用
- [ ] **包装**：包装材料 + 包装工
- [ ] **物流**：往返物流费用
- [ ] **货损**：预留损耗（通常3-5%）

### 常用供应商参考价
| 工序 | 参考单价 | 供应商 |
|------|---------|--------|
| 复合 | 咨询宋总 | 弘辉复合 |
| 激光 | 0.5-0.8元/个 | 刘旭（杨总）|
| 绣花 | 0.4-0.7元/个 | 朱师傅（杨总）|
| 车缝 | 1.5-3元/个 | 冯杰、邓总、赵总 |
| 充棉+手工 | 2-3元/个 | 张总、徐凯 |
| 物流 | 0.2-0.3元/个 | - |

## 五、报价检查项

- [ ] **成本汇总**：材料 + 加工 + 物流 + 其他
- [ ] **利润比例**：通常10-20%
- [ ] **付款方式**：订金比例（通常30%）
- [ ] **交货日期**：预计交货时间
- [ ] **含税价**：是否需要含税
- [ ] **开票类型**：1%普票/3%普票/专票

## 六、报价格式规范

```
【材料成本】
材料1：用量 × 单价 = 金额（损耗说明）
材料2：用量 × 单价 = 金额（损耗说明）
...
材料成本合计：XXX元（X.XX元/个）

【加工成本】
工序1：单价 × 数量 = 金额
工序2：单价 × 数量 = 金额
...
加工成本合计：XXX元（X.XX元/个）

【成本汇总】
总成本：XXX元
成本单价：X.XX元/个

【报价建议】
建议报价（X%利润）：X.XX元/个
合计金额：XXX元
订金30%：XXX元
尾款70%：XXX元
```

## 七、常见漏项提醒

### 容易忘记的费用
1. **损耗**：忘记加10%或5%损耗
2. **物流**：漏掉往返物流费用
3. **包装**：漏掉包装材料和包装工
4. **质检**：漏掉质检费用
5. **价格类型**：混淆散剪价和整卷价

### 必须询问的关键信息
1. 产品名称、规格、数量
2. 材料清单（主料、辅料、配件）
3. 材料单价（散剪/整卷）
4. 是否需要复合、激光、绣花
5. 车缝、充棉、手工单价
6. 物流费用
7. 其他费用（包装、质检等）

## 八、样品管理表录入规范

### 创建时机
估价完成后立即录入

### 必填字段
- 样品日期
- 客户名称
- 产品名称
- 材料清单
- 材料成本
- 估算价格
- 样品状态（初始：打样中）

### 后续更新
- 打样完成 → 更新状态为"已完成"
- 客户确认价格 → 更新"客户确认价格"、状态改为"客户确认"
- 生成正式订单 → 关联订单字段

## 九、注意事项

1. **每次估价必须按检查清单逐项确认**，避免漏项
2. **材料单价必须明确是散剪价还是整卷价**
3. **损耗比例固定**：布料10%、配件5%、棉0%
4. **加工工序必须确认完整**：复合→激光→绣花→车缝→充棉→手工→质检→包装
5. **报价必须包含**：成本价、建议报价、订金、尾款
6. **估价完成后立即录入样品管理表**
